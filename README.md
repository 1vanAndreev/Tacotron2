# Tacotron2

Учебный проект по синтезу речи на основе нейросетевой архитектуры **Tacotron2**.

## О проекте

Этот репозиторий посвящён изучению и практической реализации синтеза речи с использованием модели **Tacotron2**.  
В рамках проекта рассматривается преобразование текста в мел-спектрограммы и дальнейшая генерация аудио.

Проект использовался как практическая база для изучения:
- нейросетевого синтеза речи,
- работы с аудиоданными,
- подготовки данных для TTS,
- запуска и тестирования модели Tacotron2,
- интеграции синтеза речи в прикладной сценарий.

## Цель

Цель проекта — разобраться в работе современных TTS-моделей и получить практический опыт в области:
- deep learning,
- speech synthesis,
- обработки аудиосигналов,
- Python-разработки.

## Стек

- **Python**
- **PyTorch**
- **Tacotron2**
- **NumPy**
- **Librosa**
- **Matplotlib**
- **Jupyter Notebook** / Python scripts

## Что реализовано

В проекте могут использоваться следующие этапы:
- подготовка текстовых и аудиоданных,
- предобработка данных,
- запуск модели Tacotron2,
- генерация мел-спектрограмм,
- преобразование результатов в аудио,
- тестирование качества синтеза.
